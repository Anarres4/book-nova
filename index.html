Questions fréquentes

Oui, Book Nova est disponible gratuitement avec certaines limitations.

Version gratuite

  • Jusqu'à 8 livres ajoutés par jour
  • Maximum 150 livres AJOUTÉS au total
  • Jusqu'à 3 prêts/emprunts simultanés
  • Calendrier de lecture
  • Accès aux badges de base

Version premium

  • Ajout illimité de livres par jour
  • Livres terminés illimités
  • Prêts et emprunts illimités
  • Statistiques de lecture de base (par mois/par an)
  • Statistiques détaillées avancées
  • Accès à tous les badges et médailles premium

Toutes les fonctionnalités principales restent accessibles gratuitement !

La connexion internet n'est requise que pour certaines fonctionnalités.

Vous pouvez annuler votre période d'essai directement depuis votre store.

Sur Android

  1. Ouvrez l'application Google Play Store
  2. Appuyez sur votre photo de profil en haut à droite
  3. Sélectionnez Paiements et abonnements > Abonnements
  4. Trouvez BookNova dans la liste
  5. Appuyez sur Annuler l'abonnement
  6. Suivez les instructions pour confirmer

Sur iOS (iPhone/iPad)

  1. Ouvrez l'application Réglages de votre appareil
  2. Appuyez sur votre nom en haut de l'écran
  3. Sélectionnez Abonnements
  4. Trouvez BookNova dans la liste
  5. Appuyez sur Annuler l'abonnement ou Annuler l'essai gratuit
  6. Confirmez l'annulation

Note : Vous conservez l'accès aux fonctionnalités premium jusqu'à la fin de votre période d'essai.

Important : L'annulation doit être effectuée au moins 24 heures avant la fin de la période d'essai pour éviter d'être facturé.

Vous pouvez exporter et importer toutes vos données.

Ce qui est exporté :

  • Tous vos livres
  • Vos badges
  • Vos prêts
  • Vos emprunts

Exporter

  1. Allez dans Paramètres
  2. Cliquez sur "Exporter mes données"
  3. Un fichier JSON sera créé

Importer

  1. Allez dans Paramètres
  2. Cliquez sur "Importer mes données"
  3. Sélectionnez votre fichier

Modes d'import :

  • Remplacer : efface les données existantes
  • Fusionner : ajoute sans doublons

Sauvegarde automatique

L'application effectue une sauvegarde automatique hebdomadaire sur votre appareil.
